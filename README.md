# Nokia 3210 Snake Game

A web-based recreation of the classic Snake game as it appeared on the Nokia 3210, rendered inside a faithful replica of the iconic phone.

## Concept

The website displays a pixel-perfect Nokia 3210 phone on screen. Within the phone's monochrome LCD display, users can play the original Snake game — matching the exact look and feel of the 1999 experience.

## Key Features

- **Nokia 3210 phone replica** — the full phone body rendered in the browser, including the curved shell, speaker grille, keypad, and antenna stub
- **Authentic LCD screen** — greenish-yellow monochrome display with the characteristic pixel grid and slight ghosting effect
- **Original Snake gameplay** — single-pixel snake, dot food, bordered play area, and the same movement speed and growth behavior as the Nokia version
- **Classic UI elements** — score display, "Game Over" screen, and menu styled exactly like the Nokia 3210 interface using the original bitmap font
- **Keypad controls** — on-screen keypad buttons (2/4/6/8 or arrow keys) for directional input, matching how the game was originally played
- **Sound effects** — optional retro beep sounds matching the Nokia piezo speaker

## Tech Stack

- TypeScript
- HTML5 Canvas for phone and game rendering
- CSS for page layout and phone styling

## Goals

1. Pixel-accurate recreation of the Snake game from the Nokia 3210
2. Visually convincing Nokia 3210 phone shell surrounding the game screen
3. Responsive design — looks good on desktop and mobile
4. Simple, lightweight, no external game frameworks
